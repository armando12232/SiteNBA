Política de Privacidade
Última atualização: abril de 2026  ·  Versão 1.0
Resumo: Coletamos apenas o necessário para operar o serviço. Não vendemos seus dados. Você pode solicitar exclusão a qualquer momento. Esta política cumpre a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).
1. Quem somos

StatCast BR é uma plataforma de análise de estatísticas esportivas e props de apostas, operada como projeto independente com sede no Brasil.

Para dúvidas sobre privacidade, entre em contato: armandoluiz500@gmail.com

2. Dados que coletamos

Coletamos as seguintes categorias de dados pessoais:

  • Dados de cadastro: endereço de e-mail e senha (armazenada em hash via Supabase Auth)
  • Dados de assinatura: plano contratado, status e datas via Stripe (não armazenamos dados de cartão — processados diretamente pela Stripe)
  • Dados de uso: logs de acesso, erros e interações para melhoria do serviço
3. Como usamos seus dados
  • Autenticar seu acesso à plataforma
  • Processar e gerenciar sua assinatura
  • Enviar e-mails transacionais (confirmação de conta, senha)
  • Detectar e prevenir fraudes e abusos
  • Melhorar funcionalidades com base em dados agregados e anonimizados

Não usamos seus dados para publicidade de terceiros nem os compartilhamos com parceiros comerciais.

4. Base legal (LGPD)

O tratamento dos seus dados se baseia em:

  • Execução de contrato (Art. 7º, V): necessário para prestar o serviço contratado
  • Legítimo interesse (Art. 7º, IX): segurança, prevenção de fraude e melhoria do serviço
  • Consentimento (Art. 7º, I): para comunicações de marketing, quando aplicável
5. Com quem compartilhamos
  • Supabase: banco de dados e autenticação (servidores na AWS)
  • Stripe: processamento de pagamentos (PCI-DSS compliant)
  • Vercel: hospedagem da aplicação

Todos os subprocessadores são contratualmente obrigados a proteger seus dados conforme padrões equivalentes a esta política.

6. Seus direitos (LGPD)

Nos termos da LGPD, você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou desatualizados
  • Solicitar anonimização, bloqueio ou eliminação de dados desnecessários
  • Portabilidade dos dados a outro fornecedor
  • Revogar consentimento a qualquer momento
  • Solicitar exclusão completa da conta e dados

Para exercer qualquer direito, envie e-mail para armandoluiz500@gmail.com com o assunto "LGPD — [seu direito]". Respondemos em até 15 dias úteis.

7. Retenção de dados
  • Dados de conta: mantidos enquanto a conta estiver ativa
  • Após exclusão da conta: eliminados em até 30 dias (exceto obrigações legais fiscais)
  • Logs de acesso: 90 dias
8. Segurança
  • Tráfego criptografado via HTTPS/TLS
  • Senhas armazenadas em hash bcrypt (nunca em texto puro)
  • Row Level Security ativo no banco de dados
  • Acesso a dados de produção restrito
9. Cookies

Utilizamos apenas cookies essenciais para manter a sessão autenticada. Não utilizamos cookies de rastreamento ou publicidade de terceiros.

10. Alterações

Esta política pode ser atualizada. Notificaremos por e-mail em caso de alterações materiais. O uso contínuo após notificação implica aceitação.
